porepy.numerics.vem.vem_source module

Discretization of the source term of an equation tailored for a dual (flux-pressure) system. The sources are assigned to the rows starting from sd.num_faces, that is, to those rows in the saddle point system that represents conservation.

class DualScalarSource(keyword='flow')[source]

Bases: Discretization

Discretization of the integrated source term int q * dx over each grid cell.

All this function does is returning a zero lhs and rhs = - param.get_source.keyword in a saddle point fashion.
